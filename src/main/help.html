OneLog

Version 1.0.0

Getting Started

OneLog helps working professionals log daily activities, track tasks with time, manage projects, and plan ahead, all in a lightweight, offline desktop app. Your data never leaves your machine. Press Alt+K to open the Command Palette anytime for quick access to any feature.

Features

Click any section below to expand details.

📝 Daily Log

Write a free-form summary of your workday: meetings, decisions, accomplishments, blockers, or anything you want to remember.

  • Uses a rich text editor with full formatting: bold, italic, underline, strikethrough, headings (H2/H3), bullet & numbered lists, blockquotes, code blocks, and horizontal rules.
  • Insert images using the 🖼 toolbar button. Images are saved as local attachments and can be resized by dragging.
  • Backlinks: Type [[2026-01-15]] to create a clickable cross-reference to another date's entry.
  • Auto-saves after 800ms of inactivity, so there's no need to hit Save manually.
  • Force save anytime with Alt+S.
  • Content is stored per date. Navigate away and come back, your log is preserved.
💡 Tip: Use headings to structure your log (e.g., "Morning Stand-up", "Afternoon Deep Work") for easier scanning later.
✅ Task Tracking

Add tasks for the current day, track completion, and stay on top of your to-do list.

  • Add tasks using the input field at the bottom. Press Enter or click "+ Add".
  • Check off tasks by clicking the checkbox. Completed tasks show a strikethrough.
  • Edit text by double-clicking on any task name.
  • Drag to reorder tasks by grabbing the ⠿ handle on the left.
  • Delete a task with the ✕ button (appears on hover).
  • Assign a project via the dropdown that appears on hover, or select one when adding a new task.
  • Task count summary (e.g., "3/5 done") is shown in the section header.
⏱️ Smart Time Tracking

Track how long you spend on each task with a smart timer that automatically manages task status and prevents runaway time.

  • Start a timer by clicking the ▶ button on any task. A live counter ticks in real time.
  • Single-task focus: Starting a timer automatically stops any other running timer, so only one task runs at a time.
  • Auto-status: In Progress. Starting a timer automatically moves the task to "In Progress" on the Kanban board. No need to drag the card manually.
  • Stop the timer with the ⏹ button on the task or from the global timer indicator in the header bar.
  • Auto-stop on completion: Checking a task as done (checkbox or dragging to "Done" on Kanban) automatically stops its running timer and saves the elapsed time.
  • Manual entry: Click the ✏️ icon (on hover) to type time manually.
  • Active indicator: A pulsing green dot and live counter appear next to the running task, plus a green badge in the header showing the task name and elapsed time.
  • Total tracked time across all tasks is shown at the top of the task section.

Overnight / stale timer protection:

  • Forgot to stop a timer before closing your laptop? No worries, the app automatically caps timers at 8 hours.
  • When the app starts, any timer that has been running longer than 8 hours is stopped and credited with the maximum 8 hours.
  • The same 8-hour cap applies when you manually stop a runaway timer or when a task is marked complete.
💡 Tip: The timer persists in the database. You can close and reopen the app, and if the timer has been running for less than 8 hours it continues counting from where it left off.

Manual time input formats:

  • 2h 30m (hours and minutes)
  • 1h (hours only)
  • 45m (minutes only)
  • 1:30 (hours:minutes)
  • 90 (plain number treated as minutes)
📁 Projects

Organize tasks under color-coded projects for better categorization.

  • Open "Manage Projects" in the sidebar to create, rename, recolor, or delete projects.
  • Each project gets a color badge that appears on assigned tasks.
  • Assign a project to a task via the dropdown selector on hover, or choose one when adding a new task.
  • Deleting a project removes the association from tasks but does not delete tasks themselves.
🏷️ Tags & Labels NEW

Categorize daily entries with color-coded tags for quick filtering and organization.

  • Manage tags from the "Tags" section in the sidebar to create, rename, recolor, or delete tags.
  • Tag entries using the tag bar above the Daily Log. Click "+ Tag" to assign existing tags or create new ones.
  • Each tag has a custom color and displays as a small pill badge on the entry.
  • Tags can be used as search filters in the advanced search.
🎯 Tomorrow's Plan & Yesterday's Plan

Write down priorities and intentions for the next day at the end of each workday.

  • Uses the same rich text editor as the daily log with full formatting and image insertion.
  • Great for preparing morning stand-ups and ensuring nothing slips through the cracks.

Yesterday's Plan NEW connects planning to action:

  • When you open any day, the previous working day's Tomorrow's Plan is shown in a collapsible amber section above the Daily Log.
  • Each plan item has a "+ Task" button (on hover) to instantly convert it into a task for today.
  • "Convert all to tasks" to bulk-create tasks from every plan item in one click.
  • Converted items are shown as struck-through so you can track what was added.
  • Click the date link in the header to navigate back to the source day.
💡 Tip: Write tomorrow's plan as a bullet list. Each bullet becomes a convertible item the next morning.
🚧 Blockers NEW

Record impediments and things you're stuck on. Blockers you write here are automatically included in the Standup Generator output.

  • Uses the same rich text editor as the daily log with full formatting support.
  • Located between the Task List and Tomorrow's Plan sections.
  • Write one blocker per line for best standup output. Each line becomes a separate bullet in the generated standup message.
  • If the Blockers field is empty, the standup message shows "none" for blockers.
  • Auto-saves along with the rest of the entry (800ms debounce).
💡 Tip: Keep blockers updated throughout the day. When you generate your standup, they're automatically included.
📋 Carry-Over Tasks

Never lose track of unfinished work. The app offers to bring incomplete tasks forward.

  • When you navigate to a new day with no tasks, a banner appears if there are incomplete tasks from the previous working day.
  • Carry-over is weekend and holiday aware and skips weekends and marked holidays to pull tasks from the last actual working day.
  • Click "Carry over" to copy those tasks to the current day.
  • The original tasks remain on the previous day (they are not moved, they are copied).
📅 Calendar & Holidays NEW

Navigate dates easily and manage holidays directly from the calendar.

  • Arrow buttons (← →) in the header move one day forward or back.
  • Calendar picker: Click the 📅 icon to open a monthly calendar. Dates with entries show a blue dot.
  • Weekend highlighting: Saturdays and Sundays are visually muted in the calendar.
  • Holiday management: Right-click any date in the calendar to mark it as a holiday (with a label) or remove an existing holiday. Holidays show as a red dot.
  • Workload heatmap: Days are color-coded with green intensity based on task count (see the Workload Heatmap section below for details).
  • "Today" button appears when you're not on the current date. Click it to jump back.
  • Sidebar date list: Click any date in the sidebar to jump directly to that entry.
  • Keyboard: Alt+← / Alt+→ for quick navigation.
💡 Tip: Holidays affect task carry-over. If Friday's tasks are incomplete and Monday is a holiday, carry-over will still pull from Friday on Tuesday.
📊 Kanban Board NEW

View all your tasks across dates in a visual Kanban board with three columns.

  • Switch to Kanban view using the board icon in the sidebar or via the Command Palette.
  • Tasks are organized into To Do, In Progress, and Done columns.
  • Drag and drop tasks between columns to update their status.
  • Smart status sync: Starting a timer auto-moves a task to "In Progress". Dragging to "Done" auto-stops any running timer.
  • Navigate to day: Alt+Click any card (or click the date label) to jump to that task's day in the diary view.
  • Filter by project using the dropdown at the top to focus on specific work.
  • Each card shows the task text, project badge, date, and time tracked.
  • Toggle task completion directly from the Kanban card.
📈 Statistics Dashboard NEW

Visualize your productivity with charts and metrics.

  • Switch to Stats view using the chart icon in the sidebar or via the Command Palette.
  • Completion rate: See your overall task completion percentage in a ring chart.
  • Streak tracker: Current streak, longest streak, and total active days.
  • Time per project: Horizontal bar chart showing hours spent on each project.
  • Weekly completions: Line chart tracking how many tasks you complete each week.
  • Date range selector: Filter statistics by Last 7 days, 30 days, 90 days, or All Time.
📄 Task Templates NEW

Save frequently-used task sets as templates and apply them with one click.

  • Click the template menu below the task list to manage templates.
  • Save current tasks as template: Capture your current day's task list as a reusable template.
  • Apply a template: Instantly add a set of predefined tasks to the current day.
  • Recurring templates: Mark a template as "recurring" and its tasks will be automatically added to each new day.
  • Edit or delete templates from the template menu.
💡 Tip: Create a "Daily Standup" recurring template with tasks like "Check emails", "Review PRs", "Update Jira" to automate your morning routine.
🌙 Dark Mode NEW

Switch between light, dark, and system-matched themes.

  • Find the theme toggle at the bottom of the sidebar.
  • Click to cycle through: Light → System → Dark.
  • System mode automatically follows your OS dark mode preference and updates in real time.
  • Your preference is saved and remembered across sessions.
🔍 Search

Find past entries quickly with full-text search.

  • Use the search bar in the sidebar to search across all daily logs and tomorrow's plans.
  • Results show matching dates with snippets. Click any result to navigate to that day.
📋 Standup Generator NEW

Generate a formatted standup message from your diary data with one click, perfect for pasting into Slack, Teams, or email.

  • Click the clipboard icon (📋) in the header bar to generate your standup message.
  • The standup message includes:
    • Yesterday's completed tasks, pulled from the previous working day's task list
    • Key highlights from yesterday's daily log
    • Today's plan, including pending tasks for the current day
    • Blockers pulled from today's Blockers field (shows "none" if empty)
  • The message is copied to your clipboard instantly. Just paste it wherever you need it.
  • A "Standup copied!" tooltip appears briefly to confirm success.
  • Also available via the Command Palette (Alt+K → "Generate Standup").
💡 Tip: Fill in your tasks and daily log at the end of each day. Next morning, one click gives you a ready-made standup message.
🔗 Entry Backlinks NEW

Create clickable cross-references between diary entries using date links.

  • Type [[2026-01-15]] in any Daily Log or Tomorrow's Plan editor to create a backlink.
  • The text auto-converts into a blue date chip showing the short date (e.g., "Jan 15") with a calendar icon.
  • Click the chip to instantly navigate to that date's entry.
  • "Referenced by" panel: When other entries mention the current date, a panel appears below Tomorrow's Plan showing all entries that link to today.
  • Click any date in the "Referenced by" list to navigate to that entry.
  • Works in both Daily Log and Tomorrow's Plan editors.
💡 Tip: Use backlinks to connect related entries. Reference the date of an incident report, a sprint planning session, or a follow-up action item.
📊 Workload Heatmap NEW

See your workload at a glance with a GitHub-style heatmap on the calendar.

  • Open the calendar picker (📅 icon) where days are color-coded based on task count.
  • Color intensity indicates workload:
    • No color: no tasks
    • Light green: 1–2 tasks
    • Medium green: 3–4 tasks
    • Darker green: 5–6 tasks
    • Darkest green: 7+ tasks
  • Hover over any day to see the exact task count in a tooltip (e.g., "5 tasks").
  • Works in both light and dark modes with appropriate color adjustments.
💡 Tip: Use the heatmap to spot overloaded days and balance your workload across the week.
📤 Import & Export

Back up your data or transfer it between machines using the Tools menu.

  • Export as JSON: Menu bar → Tools → Export as JSON. Creates a structured file with all entries, tasks, projects, tags, holidays, and templates.
  • Export as CSV: Menu bar → Tools → Export as CSV. Creates a spreadsheet-ready file with one row per task per day.
  • Export as Markdown: Menu bar → Tools → Export as Markdown. Saves the current day's log as a .md file.
  • Export as PDF: Menu bar → Tools → Export as PDF. Saves the current view as a PDF document.
  • Import from JSON: Menu bar → Tools → Import from JSON. Imports entries, tasks, projects, tags, and holidays from a previously exported JSON file.
💡 Tip: Import merges data and won't delete your existing entries. Entries with the same date will be updated.
⌨️ Command Palette NEW

Quickly access any action or navigate anywhere with a keyboard-driven command palette.

  • Press Alt+K to open the Command Palette.
  • Type to fuzzy-search through all available commands.
  • Use arrow keys to navigate and Enter to execute.
  • Available commands include: view switching (Diary, Kanban, Stats), navigation (Today, Previous/Next day), section focus (Daily Log, Tasks, Plan), generate standup, export, and more.
🗑️ Reset All Data

Start fresh by deleting all entries, tasks, projects, tags, holidays, templates, and settings.

  • Find the "Reset All Data" button at the bottom of the sidebar.
  • A confirmation modal appears with a 5-second countdown before the delete button becomes active to prevent accidental deletions.
  • This action is irreversible. Consider exporting your data first via Tools → Export.

Data & Privacy

📂 Where is my data stored?

All data is stored locally in a SQLite database file at:

%APPDATA%\onelog\onelog.db

Image attachments are stored in:

%APPDATA%\onelog\attachments\

No data is sent to any server. The app works 100% offline with no internet connection required.

🔒 Is my data secure?

Your data stays entirely on your local machine. The app does not connect to external servers, analytics services, or cloud storage. Your diary is private by design.

To keep your data safe, consider:

  • Regularly exporting a backup via Tools → Export as JSON.
  • Storing the export file in a secure location (e.g., encrypted drive, cloud backup folder).
💻 Can I move my data to another computer?

Yes! Use the Import & Export feature:

  • On the old machine: Tools → Export as JSON
  • Transfer the JSON file to the new machine
  • On the new machine: Tools → Import from JSON

Alternatively, you can copy the SQLite database file directly from %APPDATA%\onelog\.

🔄 What happens when I update the app?

Your existing data is always preserved during updates. The app automatically migrates your database to the new structure, adding new tables and columns as needed without touching your existing entries, tasks, or projects.

Frequently Asked Questions

The app already has sample data. Where did it come from?

On first launch, the app is seeded with sample data (a fictional "Data Engineer on Finance Project" story spanning about a month) so you can explore features immediately. When you're ready to start fresh, use the Reset All Data button in the sidebar.

Can I run multiple timers at once?

No. Starting a timer on a task automatically stops any other running timer. This enforces single-task focus to keep your time tracking accurate.

What happens if I close the app while a timer is running?

The timer start timestamp is saved in the database. If you reopen the app within 8 hours, the timer picks up from where it left off and continues counting. If more than 8 hours have passed (e.g., you left the app open overnight), the timer is automatically stopped and capped at 8 hours to prevent inflated tracking.

Does starting a timer change my task's Kanban status?

Yes. Starting a timer automatically moves the task to "In Progress" on the Kanban board. Similarly, completing a task (checkbox or Kanban drag to "Done") automatically stops the running timer.

Can I edit entries from past dates?

Yes. Navigate to any past date using the arrows, calendar picker, or sidebar, and edit the daily log, tasks, or tomorrow's plan freely. Changes auto-save.

How do I mark a holiday?

Open the calendar picker (📅 icon) and right-click on any date. Enter a label (e.g., "Diwali", "Christmas") and click "Add Holiday". Holidays are shown with a red dot in the calendar and are skipped during task carry-over.

How do recurring templates work?

When a template is marked as recurring, its tasks are automatically added whenever you navigate to a new day that has no tasks yet. This happens only once per day. If you delete the auto-added tasks, they won't reappear unless you apply the template manually.

How do I access Import/Export?

Use the Tools menu in the application menu bar (top of the window). You'll find "Import from JSON", "Export as JSON", "Export as CSV", "Export as Markdown", and "Export as PDF" there.

How do I create a backlink to another date?

In the Daily Log or Tomorrow's Plan editor, type [[ followed by a date in YYYY-MM-DD format and then ]]. For example, typing [[2026-01-15]] will auto-convert into a clickable blue chip showing "Jan 15". Click the chip to navigate to that date's entry.

How does the standup generator work?

Click the clipboard icon (📋) in the header bar. The app pulls your previous working day's completed tasks and log highlights, plus today's pending tasks and blockers from the Blockers field, and formats them into a standup message. The message is copied to your clipboard instantly. Just paste into Slack, Teams, or email.

What do the green colors on the calendar mean?

The calendar uses a workload heatmap where days with more tasks show darker green backgrounds. This gives you a quick visual overview of how busy each day was. Hover over any day to see the exact task count.

Is the app available on macOS and Linux?

Yes! The app can be built for Windows (NSIS installer), macOS (DMG for Intel and Apple Silicon), and Linux (AppImage and .deb). Check the project repository for platform-specific build instructions.
